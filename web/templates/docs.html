Overview

Yuyan is a powerful notification service that enables you to send messages to multiple messaging platforms through a unified API. It currently supports DingTalk and Telegram, with a clean web interface for configuration and management.

Features

  • Multi-platform support (DingTalk, Telegram)
  • Unified API for sending notifications
  • Bot configuration management
  • Message history and delivery status tracking
  • Web interface for easy management
  • Support for multiple database types

API Usage

Message Formats

The API supports three message formats:

  • text - Plain text messages without formatting
  • html - Messages with HTML formatting and mentions
  • markdown - Messages with Markdown formatting

    if err != nil {
        panic(err)
    }

    client := &http.Client{}
    resp, err := client.Do(req)
    if err != nil {
        panic(err)
    }
    defer resp.Body.Close()
    
    // Read and print response
    var result map[string]interface{}
    json.NewDecoder(resp.Body).Decode(&result)
    fmt.Println(result)
}

User Mentions

You can mention users in three ways:

  • @username - Mention specific users
  • @all - Mention all users in a group
  • @phone_number - Mention users by phone number (DingTalk only)

import requests

def send_message(content):
    url = "http://localhost:8090/api/messages"
    payload = {
        "bot_id": 4,
        "content": content,
        "format": "html"
    }
    headers = {
        "Content-Type": "application/json"
    }
    response = requests.post(url, json=payload, headers=headers)
    return response.json()

# Mention specific users
response = send_message("Hey @wolf1688, please check this! cc: @learnjie")
print(response)

# Mention all users
response = send_message("Important announcement @all: Team meeting at 3 PM!")
print(response)

Successful Response

    "status": "sent"
  }

Error Response
{
  "code": 400,
  "message": "Invalid request",
  "error": "Bot not found or inactive"

Bot Configuration

DingTalk Configuration

To configure a DingTalk bot, you'll need:

  • Webhook URL from DingTalk custom robot configuration
  • Optional secret for signing requests

Telegram Configuration

To configure a Telegram bot, you'll need:

  • Bot token from the BotFather
  • Chat ID where messages will be sent
